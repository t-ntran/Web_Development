ERSP 2021-2022

During the school year 2021-2022, I conducted research to examine if a live programming environment would help and encourage programmers to write and test their code. Specifically, the live programming environment that I was investing was Projection Boxes (PB). My peers and I developed a testing framework called PBUnit that would work within PB.

Summer Research 2022

During the summer of 2022, I did a follow up project to my ERSP project. I started to investing if a live programming environment would help programmers make sense of AI generated code, specifically. code generated by Copilot. I focused mostly on researching whether live programming would help programmers trust Copilot's code more or if it would help programmers debug Copilot's code.

ZookSeeker

I worked in a group to develop an app in Android Studio. Our app, ZookSeeker, was built to help users plan their day and navigate around the San Diego Zoo. Working on this app taught me how to build out fragments and the imporantace of the MVP model.
